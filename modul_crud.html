Program SQL Database

1. Membuat Database

CREATE DATABASE IF NOT EXISTS latihan_crud;
USE latihan_crud;
  • CREATE DATABASE IF NOT EXISTS latihan_crud;

    ‚ûù Membuat database baru dengan nama latihan_crud.

    ‚ûù IF NOT EXISTS artinya: kalau sudah ada, jangan error, cukup lewati.

  • USE latihan_crud;

    ‚ûù Memilih database latihan_crud sebagai database aktif.

    ‚ûù Semua perintah SQL berikutnya akan dijalankan di database ini.

2. Membuat Tabel

CREATE TABLE IF NOT EXISTS barang (
  id INT AUTO_INCREMENT PRIMARY KEY,
  nama VARCHAR(100),
  harga INT
) ENGINE=InnoDB;
  • CREATE TABLE IF NOT EXISTS barang

    ‚ûù Membuat tabel baru dengan nama barang.

    ‚ûù IF NOT EXISTS = kalau tabel barang sudah ada, MySQL tidak error.

  • id INT AUTO_INCREMENT PRIMARY KEY

    ‚ûù Kolom id bertipe integer.

    ‚ûù AUTO_INCREMENT ‚Üí nilainya otomatis bertambah setiap kali menambah data baru.

    ‚ûù PRIMARY KEY ‚Üí menjadi kunci unik untuk tiap baris.

  • nama VARCHAR(100)

    ‚ûù Kolom nama bertipe varchar (teks pendek) maksimal 100 karakter.

    ‚ûù Digunakan untuk menyimpan nama barang.

  • harga INT

    ‚ûù Kolom harga bertipe integer (bilangan bulat).

    ‚ûù Digunakan untuk menyimpan harga barang.

  • ) ENGINE=InnoDB;

    ‚ûù Menentukan mesin database yang dipakai adalah InnoDB.

    ‚ûù InnoDB mendukung transaksi, foreign key, dan lebih stabil untuk aplikasi modern.

3. Alur sederhananya

  1. Pertama kamu buat database bernama latihan_crud.
  1. Di dalamnya, dibuat tabel barang.
  1. Tabel ini punya 3 kolom:
    • id (nomor otomatis)
    • nama (teks nama barang)
    • harga (angka harga barang)
  1. Tabel siap dipakai oleh PHP CRUD kamu untuk tambah, lihat, ubah, hapus barang.

üëâ Jadi file produk.sql ini adalah skema awal untuk latihan CRUD. Tanpa menjalankan file ini di MySQL/phpMyAdmin, tabel barang tidak akan ada, sehingga index.php, tambah.php, edit.php, dan hapus.php tidak bisa berjalan.

Materi Bootstrap

1. Apa itu Bootstrap?

  • Bootstrap adalah framework CSS + JavaScript yang sudah berisi kumpulan style, komponen, dan layout siap pakai.
  • Fungsinya: mempercepat membuat tampilan website yang rapi, modern, dan responsif tanpa harus menulis CSS dari nol.
  • Versi terbaru sekarang adalah Bootstrap 5 (yang kamu pakai di header.php lewat CDN).

2. Kenapa Bootstrap dipakai?

‚úÖ Responsif otomatis ‚Üí tampilan menyesuaikan layar HP, tablet, laptop.

‚úÖ Komponen siap pakai ‚Üí tombol, navbar, form, tabel, alert, modal, dll.

‚úÖ Mudah dipelajari ‚Üí cukup pakai class di elemen HTML.

‚úÖ Konsisten ‚Üí hasil desain seragam, tidak perlu atur ulang style tiap halaman.

3. Cara Memakai Bootstrap

Ada dua cara:

  1. CDN (paling mudah) ‚Äì cukup tambahkan link CSS & JS dari internet:
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>
  1. Download file ‚Äì unduh dan simpan CSS/JS Bootstrap di folder project (butuh setting manual).

Di CRUD kamu, kamu sudah pakai CDN.

4. Grid System (Layout Utama Bootstrap)

  • Bootstrap pakai sistem grid 12 kolom.
  • Kamu atur lebar konten dengan class seperti col-, col-md-, col-lg-.

Contoh:

<div class="container">
  <div class="row">
    <div class="col-md-8">Konten utama (8 kolom)</div>
    <div class="col-md-4">Sidebar (4 kolom)</div>
  </div>
</div>

‚û°Ô∏è Di layar besar, konten terbagi 8 + 4 kolom. Di layar kecil, otomatis turun ke bawah (responsif).

5. Komponen Populer Bootstrap

  1. Button

<button class="btn btn-danger">Delete</button>

    ‚ûù Warna & style tombol langsung jadi.

  • Margin & padding: m-3 (margin 1rem), p-2 (padding 0.5rem).

Contoh:

Program Koneksi PHP

kode koneksi.php kamu itu sudah benar dan cukup untuk latihan CRUD sederhana dengan mysqli.

Agar lebih jelas, aku kasih beberapa catatan kecil supaya kamu lebih paham fungsinya:

<?php
// Membuat koneksi ke MySQL
$koneksi = mysqli_connect("localhost", "root", "", "latihan_crud");

// Cek apakah koneksi berhasil
if (!$koneksi) {
    die("Koneksi gagal: " . mysqli_connect_error());
}

// echo "Koneksi berhasil!"; // (opsional) bisa dipakai untuk tes
?>

Penjelasan baris per baris:

  1. mysqli_connect("localhost", "root", "", "latihan_crud");
    • "localhost" ‚Üí alamat server database (default: localhost / 127.0.0.1).
    • "root" ‚Üí username default MySQL di XAMPP/Laragon.
    • "" ‚Üí password default (kosong). Kalau MySQL-mu ada password, isi di sini.
    • "latihan_crud" ‚Üí nama database yang sudah kamu buat.
  1. if (!$koneksi)

    Mengecek apakah koneksi gagal. Jika iya, tampilkan error.

  1. die("Koneksi gagal: " . mysqli_connect_error());

    Menghentikan program dan menampilkan pesan error jika koneksi tidak berhasil.

üí° Tips:

  • Simpan file ini di folder project kamu dengan nama koneksi.php.
  • Di file lain tinggal tambahkan:
    <?php include "koneksi.php"; ?>

    supaya otomatis terkoneksi.

  • $errors ‚Üí array untuk menyimpan pesan error validasi.
  • $nama dan $harga ‚Üí variabel untuk menyimpan input dari form.

3. Logika saat tombol simpan ditekan

if (isset($_POST['simpan'])) {
    $nama = trim($_POST['nama'] ?? '');
    $harga = trim($_POST['harga'] ?? '');
    if ($nama === '') $errors[] = 'Nama wajib diisi';
    if ($harga === '' || (int)$harga < 0) $errors[] = 'Harga harus diisi dan tidak negatif';

    if (!$errors) {
        mysqli_query($koneksi, "INSERT INTO barang (nama, harga)
          VALUES ('".mysqli_real_escape_string($koneksi,$nama)."','".(int)$harga."')");
        header('Location: index.php?msg=Data berhasil ditambah');
        exit;
    }
}
  • if (isset($_POST['simpan'])) ‚Üí cek apakah form sudah dikirim.
  • trim() ‚Üí menghapus spasi berlebih di awal/akhir input.
  • Validasi:
    • Jika nama kosong ‚Üí error.
    • Jika harga kosong atau negatif ‚Üí error.
  • Kalau tidak ada error:
    • Simpan data ke tabel barang dengan INSERT.
    • mysqli_real_escape_string ‚Üí mencegah SQL Injection.
    • (int)$harga ‚Üí memastikan harga angka.
    • Setelah simpan, diarahkan balik ke index.php dengan pesan sukses.

  • Jika ada error, tampilkan di alert merah.

<input type="text"   name="nama"  class="form-control" value="<?= htmlspecialchars($nama)  ?>" required>
<input type="number" name="harga" class="form-control" value="<?= htmlspecialchars($harga) ?>" required>
  • Field sudah terisi nilai lama; kalau validasi gagal, nilai yang sudah diketik tidak hilang.

Tombol:

Program Hapus PHP

Isi kode hapus.php kamu:

<?php
include "koneksi.php";
$id = (int)($_GET['id'] ?? 0);
if ($id > 0) {
  mysqli_query($koneksi, "DELETE FROM barang WHERE id=$id");
}
header('Location: index.php?msg=Data berhasil dihapus');
exit;

Penjelasan per baris

  1. include "koneksi.php";

    ‚ûù Menghubungkan file ini ke database MySQL. Jadi kita bisa menjalankan query SQL.

  1. $id = (int)($_GET['id'] ?? 0);

    ‚ûù Mengambil parameter id dari URL, contoh: hapus.php?id=3.

    ‚ûù (int) artinya dipaksa jadi angka (supaya aman & tidak bisa disusupi teks).

    ‚ûù Kalau tidak ada id, otomatis nilainya jadi 0.

  1. if ($id > 0) { ... }

    ‚ûù Cek apakah id valid (lebih besar dari 0).

    ‚ûù Kalau iya, baris di dalam if dijalankan.

  1. mysqli_query($koneksi, "DELETE FROM barang WHERE id=$id");

    ‚ûù Menjalankan perintah SQL untuk menghapus data barang berdasarkan id.

    ‚ûù Contoh kalau $id = 3, maka query-nya:

    DELETE FROM barang WHERE id=3

    ‚ûù Jadi data dengan id=3 di tabel barang akan hilang.

  1. header('Location: index.php?msg=Data berhasil dihapus');

    ‚ûù Setelah eksekusi selesai, pengguna langsung diarahkan (redirect) kembali ke index.php.

    ‚ûù Ada tambahan ?msg=Data berhasil dihapus di URL, supaya di index.php bisa tampil alert pesan sukses.

  1. exit;

    ‚ûù Menghentikan program agar tidak ada kode lain yang dijalankan setelah redirect.

Alur Singkatnya:

  • User klik tombol Hapus ‚Üí diarahkan ke hapus.php?id=...
  • hapus.php ambil id dari URL ‚Üí jalankan query DELETE ‚Üí kembali ke index.php
  • Di index.php tampil pesan: ‚ÄúData berhasil dihapus‚Äù.

Bootstrap 5 Cheat Sheet (CRUD Edition)

1. Layout & Container

  • container ‚Üí pembungkus dengan margin otomatis.
  • row ‚Üí baris untuk grid.
  • col, col-md-6, col-lg-4 ‚Üí kolom grid (total 12).

üìå Contoh:

<div class="container">
  <div class="row">
    <div class="col-md-8">Kolom 8</div>
    <div class="col-md-4">Kolom 4</div>
  </div>
</div>

2. Typography (Teks)

  • fw-bold, fw-semibold ‚Üí tebal.
  • text-center, text-start, text-end ‚Üí rata teks.
  • text-muted ‚Üí abu-abu lembut.
  • h1, h2, h3 ‚Üí ukuran heading bawaan Bootstrap.

üìå Contoh:

<h3 class="fw-bold text-center text-muted">Daftar Barang</h3>

3. Buttons

  • btn btn-primary ‚Üí tombol biru.
  • btn btn-success ‚Üí tombol hijau.
  • btn btn-danger ‚Üí tombol merah.
  • btn btn-warning ‚Üí tombol kuning.
  • btn-outline-light ‚Üí tombol outline putih.
  • btn-sm ‚Üí tombol kecil.

üìå Contoh:

<button class="btn btn-success btn-sm">+ Tambah</button>

4. Forms

  • form-control ‚Üí input rapi.
  • form-label ‚Üí label input.
  • mb-3 ‚Üí jarak bawah antar form.

üìå Contoh:

<div class="mb-3">
  <label class="form-label">Nama</label>
  <input type="text" class="form-control">
</div>

5. Cards (Kotak Konten)

  • card ‚Üí komponen dasar.
  • card-header ‚Üí judul card.
  • card-body ‚Üí isi card.
  • shadow-sm ‚Üí efek bayangan.

üìå Contoh:

<div class="card shadow-sm">
  <div class="card-header fw-semibold">Tambah Barang</div>
  <div class="card-body">Isi Form</div>
</div>

6. Tables

  • table ‚Üí tabel dasar.
  • table-striped ‚Üí zebra strip.
  • table-bordered ‚Üí ada garis.
  • align-middle ‚Üí teks rata tengah vertikal.
  • table-responsive ‚Üí tabel bisa digeser di HP.

üìå Contoh:

<div class="table-responsive">
  <table class="table table-striped table-bordered align-middle">
    <thead class="table-dark">
      <tr><th>No</th><th>Nama</th><th>Harga</th></tr>
    </thead>
  </table>
</div>

7. Alerts (Pesan)

  • alert alert-success ‚Üí pesan hijau.
  • alert alert-danger ‚Üí pesan merah.
  • alert alert-info ‚Üí pesan biru.

üìå Contoh:

<div class="alert alert-success">Data berhasil ditambah</div>

8. Navbar

  • navbar navbar-dark bg-dark ‚Üí navbar gelap.
  • navbar-brand ‚Üí nama/logo aplikasi.
  • d-flex gap-2 ‚Üí tombol kanan berjejer dengan jarak.

üìå Contoh:

<nav class="navbar navbar-dark bg-dark">
  <div class="container">
    <a class="navbar-brand" href="#">Latihan CRUD</a>
    <div class="d-flex gap-2">
      <a class="btn btn-outline-light btn-sm" href="index.php">Daftar</a>
      <a class="btn btn-success btn-sm" href="tambah.php">+ Tambah</a>
    </div>
  </div>
</nav>

9. Utilities (Class Serbaguna)

  • m-3, mt-2, mb-4 ‚Üí margin (atas/bawah/samping).
  • p-3, py-4, px-2 ‚Üí padding.
  • d-flex ‚Üí aktifkan flexbox.
  • justify-content-between ‚Üí bagi kiri-kanan.
  • justify-content-center ‚Üí rata tengah.
  • align-items-center ‚Üí rata vertikal.
  • gap-2, gap-3 ‚Üí jarak antar elemen.
  • bg-light, bg-dark, bg-primary ‚Üí warna latar.

üìå Contoh:

<div class="d-flex justify-content-between align-items-center bg-light p-3">
  <span>Kiri</span>
  <span>Kanan</span>
</div>

‚ú® Kesimpulan:

Untuk proyek CRUD, class yang paling sering dipakai adalah:

  • btn, btn-*, btn-sm (tombol)

  • card, card-header, card-body (box konten)
  • navbar navbar-dark bg-dark (navigasi atas)
  • container, row, col-md-* (layout/grid)
