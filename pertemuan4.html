âœ¨ Cara Membuat SQL Code

  1. Hidupkan XAMPP, jika sudah aktifkan Apache dan MySQL.
  2. Masuk ke Windows Explorer (atau yang kalian kenal dengan File Manager).
  3. Buka folder instalasi XAMPP lalu masuk ke folder "htdocs".
  4. Buat folder atas nama kalian di dalam folder "htdocs" tadi (bisa dengan shortcut CTRL + SHIFT + N).
  5. Buka folder kalian di dalam Visual Studio Code.
  6. Buat file baru dengan format file ".sql", lalu copy kan source code dibawah, dan save
  7. Buka browser dan akses http://localhost/phpmyadmin.
  8. Pilih database yang ingin kalian gunakan, atau klik "New" untuk buat database baru.
  9. Masuk ke tab "Import" di menu phpMyAdmin.
  10. Upload file .sql yang sudah kalian buat tadi, lalu klik tombol "Go" untuk mulai proses impor.
            

-- 1. Buat dulu databasenya
CREATE DATABASE tugas_11_rpl_1;

-- 2. Gunakan database tersebut
USE tugas_11_rpl_1;

-- 3. Buat tabel anggota_perpustakaan
CREATE TABLE anggota_perpustakaan (
    ID INT PRIMARY KEY,
    Nama_Lengkap VARCHAR(100),
    NIM_ID_Anggota VARCHAR(10),
    Jenis_Kelamin ENUM('Laki-laki', 'Perempuan'),
    Tanggal_Lahir DATE,
    Alamat VARCHAR(255),
    No_Telepon VARCHAR(15),
    Email VARCHAR(100),
    Status_Anggota ENUM('Aktif', 'Tidak Aktif'),
    Tanggal_Bergabung DATE
);

-- 4. Masukkan data ke tabel
