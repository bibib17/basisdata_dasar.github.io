1. Menghidupkan MySQL Server

Pastikan MySQL sudah berjalan. Gunakan XAMPP Control Panel.

  • tekan tombol Start untuk aktifkan Apache dan MySQL di XAMPP Control Panel.

2. Membuka Command Prompt (CMD)

Langkah-langkah:

Langkah-langkah:
1. Klik tombol Start atau tekan tombol Windows di keyboard.
2. Ketik "cmd" atau "Command Prompt".
3. Klik aplikasi Command Prompt yang muncul di hasil pencarian.

Langkah-langkah:
1. Tekan tombol Windows + R secara bersamaan untuk membuka jendela Run.
2. Ketik cmd pada kotak yang muncul.
3. Tekan Enter atau klik OK.

Langkah-langkah:
1. Tekan Ctrl + Shift + Esc untuk membuka Task Manager.
2. Klik menu File > Run new task.
3. Ketik cmd lalu tekan Enter atau klik OK.

Langkah-langkah:
1. Klik kanan pada tombol Start di pojok kiri bawah layar.
2. Pilih Windows Terminal atau Command Prompt dari menu yang muncul.

3. Masuk Kedalam Direktori MySQL

Langkah-langkah:

3. Menampilkan Daftar Database

SHOW DATABASES;

4. Membuat Database Baru

CREATE DATABASE latihan1_sql;

5. Memilih Database

USE latihan1_sql;

6. Membuat Tabel Baru

CREATE TABLE siswa (
  id INT PRIMARY KEY AUTO_INCREMENT,
  nama VARCHAR(255),
  jenis_kelamin VARCHAR(255),
  umur INT
);

7. Melihat Isi Tabel

SELECT * FROM siswa;

8. Menambahkan Data ke Tabel

INSERT INTO siswa (nama, jenis_kelamin, umur)
VALUES
  ('budi', 'laki-laki', 15),
  ('ani', 'perempuan', 14),
  ('rio', 'laki-laki', 15);

9. Menampilkan Data Berdasarkan Kriteria

SELECT * FROM siswa WHERE jenis_kelamin = 'laki-laki';

10. Menghapus Data dari Tabel

DELETE FROM siswa WHERE id = 2;

11. Menambahkan Kolom Baru

ALTER TABLE siswa ADD COLUMN alamat VARCHAR(255);

12. Memperbarui Data

UPDATE siswa SET alamat = 'tembung' WHERE nama = 'budi';
